# ⚰️ CoffinTranslate

A modern translation editor and language manager for **The Coffin of Andy and Leyley**.

Create, edit, build and install community translations with an intuitive interface. No manual file copying or spreadsheet editing required.

> **This is an unofficial community project and is not affiliated with the game's developer.**

---

## ✨ Features

- 🌍 Create new translation projects
- ✏️ Modern translation editor
- 🔍 Search through dialogue, labels and items
- 💾 Automatic saving
- 📊 Translation progress tracking
- 📦 One-click language installation
- 🔄 Update existing translation projects
- 🖼 Support for translated images
- 🔤 Custom font support
- ⚡ Fast and lightweight
- 🐧 Windows & Linux support

## 🌐 Community Language Packs

Language packs created by the community are available through the application.

Simply open **Community**, choose your language and install it with a single click.

No manual copying required.

---

## 🛠 Creating a Translation

1. Create a new project.
2. Select your language.
3. Translate the English text.
4. Save your progress.
5. Build or install your translation.
6. Share it with the community.
